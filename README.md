# Radium test task

My solution to the test task for the Backend-developer vacancy from the Radium company.

## Setup

```commandline
git clone https://github.com/RolfAdolf/radium_test_task.git

cd radium_test_task

poetry install
```

## Run script

```commandline
python3 -m src.main
```

## Run tests

### Tests
```commandline
pytest
# add --cov option to generate code coverage report
```

### Check the coverage
```commandline
coverage report
```
